# student-gemini-task (템플릿)

이 폴더는 `preview (1).html` 튜토리얼에서 학생이 따라 만드는 파일들을 미리 생성해 둔 템플릿입니다.

## 포함 파일
- `GEMINI.md`
- `.gemini/skills/*/SKILL.md` (5개)
- `.gemini/commands/taskflow/*.toml` (2개)

## 빠른 시작
```bash
cp -R . ~/student-gemini-task
cd ~/student-gemini-task
```

Gemini CLI 실행 후(필요 시):
```text
/memory reload
/skills reload
/commands reload
```
